Step 2 计算特殊属性

伤害加值与体格：
伤害加值DB代表成功的近战攻击带来的伤害修正，体格Build则在近战判定时有用。
将你在第一步中分配出的力量STR和体系STZ相加，然后参考下面的表格得出这两个数值（我不会Markdown表格）

STR+STZ  DB  Build
2-64     -2   -2
65-84    -1   -1
85-124   无    0
125-164 +1d4   1
165-204 +1d6   2

健康值HP：
HP没什么好解释的。一个变量。=0就要进急救判定了。
HP = (CONp+STZ)//10 （注意整除，后面同理）
移动速度MOV：
人类固定为8。
理智值SAN：
很知名的一个变量，不多解释。理智值初始等于意志POW。
魔法值MP：
顾名思义的一个变量，蓝条。MP=POW//5
幸运值：
扔三次六面骰子（3d6，没有骰子可以卡西欧RandInt(1,6)按三下），将结果乘5得到幸运值。

首先，根据你所选择的职业，合理地（或者直接按照规则书说明，如果你的职业已经有说明的话）将如下数值覆盖（直接取代基础数值）在你所想要的技能（必选信用等级！你也不想身无分文吧）上：70，60，60，50，50，50，40，40，40 这代表着你的角色因为职业原因学习到的技能。
